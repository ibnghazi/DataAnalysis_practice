## NumPy
###### Numerical Python
: 다차원 배열을 효율적으로 다룰 수 있는 라이브러리

### ndarray
- N-dimensional Array (N-차원 배열) 
- 다차원 배열을 표현할 수 있는 객체(구조)
- 모든 요소가 <b>동일한 데이터 타입</b>을 가짐

### 패키지 설치 필요
> pip install numpy

---

## Pandas
: Python 데이터 분석 핵심 라이브러리

### Series
- 라벨링된 1차원 데이터 구조
- 데이터 + 인덱스

### DataFrame
- 라벨링된 2차원 데이터 구조
- 행/열을 구분하는 라벨(이름)이 존재

### 패키지 설치 필요
> pip install pandas
